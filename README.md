# Bezawit's Portfolio

A premium, modern portfolio built with [Next.js](https://nextjs.org/) and [Tailwind CSS](https://tailwindcss.com/).

## Features

- **Responsive Design**: Optimized for all devices, from mobile to desktop.
- **Dark Mode Support**: Seamlessly transitions between light and dark themes.
- **Interactive Sections**:
  - **About**: Professional introduction and bio.
  - **Skills**: Visual representation of technical expertise.
  - **Education**: Timeline of academic achievements.
  - **Certifications**: Verified professional credentials.
  - **Projects**: Showcase of featured work.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Structure

- `/app`: Main application routes and global styles.
- `/components`: Modular UI components organized by feature.
- `/public`: Static assets (images, icons).

## Customization

You can customize the content by editing the components in the `components/` directory. Each section uses Tailwind CSS for easy styling adjustments.
